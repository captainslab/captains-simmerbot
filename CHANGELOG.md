# Changelog

## v0.5.0 - 2026-04-08

First shareable release of the BTC Sprint Bot stack.

### Highlights

- Natural-language Discord control with `?` and mention-based routing.
- `!` shortcuts remain available as a fast fallback for direct actions.
- Portable install path with repo-relative defaults and env overrides.
- Clean onboarding docs for install, updates, and first-time setup.
- Branded README and banner so the project presents well when shared.
- Secret hygiene sweep completed and test suite passing before release.

### Notes

- Default behavior stays dry-run-first.
- BTC-only scope and deterministic risk gates remain unchanged.
- Users can keep local secrets and machine-specific settings outside the repo.
